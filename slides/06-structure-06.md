### Cabecera
#### Incluir archivos
Para incluir estilos se utiliza la etiqueta _link_.

````HTML
<!DOCTYPE html>
<html>
    <head>
        <title>Hello World</title>
        <meta charset="utf-8" />
        <meta name="description" content="This is my first page">
        <link rel="stylesheet" type="text/css" href="css/style.css">
    </head>
</html>
````
